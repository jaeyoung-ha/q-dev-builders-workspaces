📋 직무 소개

✅ 필수 요구사항
  • SQL 2년 이상 실무 경험
  • Python 또는 R을 활용한 데이터 분석 경험
  • 통계학 기초 지식 보유 (기술통계, 추론통계)
  • 데이터 시각화 도구 (Tableau, Power BI) 경험
  • 비즈니스 이해도 및 커뮤니케이션 능력
  • Excel 고급 기능 활용 능력
🎯 우대 사항
  • 머신러닝 모델링 경험 (scikit-learn, TensorFlow)
  • A/B 테스트 설계 및 분석 경험
  • 빅데이터 처리 경험 (Spark, Hadoop)
  • 클라우드 데이터 플랫폼 경험 (AWS Redshift, BigQuery)
  • 비즈니스 도메인 지식 (마케팅, 금융, 이커머스)
  • 데이터 파이프라인 구축 경험
  • 웹 분석 도구 (Google Analytics, Adobe Analytics) 경험
💼 주요 업무
  • 비즈니스 데이터 분석 및 인사이트 도출
  • KPI 모니터링 대시보드 구축 및 운영
  • A/B 테스트 설계 및 결과 분석
  • 예측 모델링 및 통계 분석

  • 분석 결과 리포트 작성 및 발표
